Get rid of WordPress another update is already in progress

== Description ==

Sometimes though there is no update in progress, you might unable to update your WordPress site to the newest version of WordPress because you got a message like "Another update is currently in progress". It is an automatic lock state to prevent WordPress from simultaneous core updates. So if you want to get rid of that message and upgrade your WordPress site without waiting that long, then follow the steps below.<a href="https://www.proy.info/wordpress-another-update-is-currently-in-progress/">Read more..</a>

== Installation ==

1. Upload the fix-another-update-in-progress folder to the /wp-content/plugins/ directory
2. Activate the Fix Another Update In Progress plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings - Fix Another Update In Progress on the menu.
